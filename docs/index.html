Добро пожаловать на сайт ED-LINK

Лицензирование программного обеспечения для юридических лиц

Компания ED-LINK является единственным официальным лицензором следующих онлайн-сервисов и всего предоставляемого ими программного обеспечения соответственно:
TrustLink.ru, SeoPult.Pro, WebArtex.ru

Продажа (передача) лицензий на программное обеспечение, как объектов авторского права, осуществляется только для юридических лиц по безналичному расчету на основании лицензионного договора, соответствующего требованиям законодательства РФ.



* Физическим лицам, желающим использовать программное обеспечение вышеуказанных сервисов, достаточно простой регистрации на сайте соответствующего проекта с подтверждением договора-оферты в режиме онлайн.

Как получить лицензию

  • Отправьте заявку на получение лицензии через форму обратной связи. В заявке укажите реквизиты вашей организации и название сервиса, лицензия которого вас интересует.
  • Дождитесь ответного письма менеджера компании Эдлинк. В письме вы получите общие инструкции по дальнейшему протоколу сделки.
  • После выполнения инструкций менеджер компании Эдлинк вышлет вам по почте двойной комплект документов для подписания. Один из подписанных комплектов нужно будет отправить обратно в Эдлинк.
  • Лицензионное соглашение вступает в законную силу после подтверждения получения подписанного комплекта документов компанией Эдлинк
